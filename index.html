Digital Stopwatch

In this project, I designed a digital stopwatch using mod-10 and mod-6 asynchronous counters using J-K flip-flops and seven-segment displays representing the digits in binary. It can count to digit 59 from 00. Using Asynchronous clock inputs we can acheive the digital stop watch of 60 seconds. This can further improved to higher versions with more no of flip flops depending upon the count and its bits.

Automatic Streetlight System

Initially, I built a Light detecting sensor in this project using a basic NAND gate and Photoresistor. Further, I implemented it for Automatic Street light control with the help of an 8051 microcontroller and BC547 Transistor implemented and simulated in Proteus software.
